# 通用技术走势分析框架：板块与个股共用

日期：2026-07-09

## 适用范围

本框架同时适用于：

- 产业重点板块
- 行业板块
- 概念板块
- 用户持仓个股
- 候选观察个股

## 核心原则

无论分析板块还是个股，都必须先判断基本走势，再判断指标信号。

判断顺序：

1. 基本走势：上涨 / 下跌 / 横盘震荡
2. 如果横盘震荡，识别箱体上沿、下沿、中轴
3. 日线 KDJ / MACD：判断短期状态
4. 周线 KDJ / MACD：判断中期趋势
5. 月线 KDJ / MACD：判断长期趋势
6. 结合大盘、板块、个股相对强弱决定操作

## 术语约定

本体系后续单独出现的 `J`、`J值`、`日线J`、`周线J`、`月线J`，均指对应周期 **KDJ 指标的 J 值**，不代表其他技术指标。

## 个股适用时的额外规则

个股技术走势不能脱离板块：

- 个股上涨，但板块下跌：判断是否有独立逻辑，否则谨慎追高
- 个股横盘，但板块上涨：判断是否弱于板块，可能是后排或资金不认可
- 个股下跌，但板块上涨：弱势信号，持仓需提高风控
- 个股调整到位，但板块未调整到位：只观察，不急于买入
- 个股与板块同时调整到位：进入重点观察区

## 个股趋势分类

### 核心均线口径

- 统一关注 `MA25 / MA60 / MA144 / MA240`，不再以MA20作为核心趋势均线。
- MA25：短中期节奏和回踩/收复确认。
- MA60：波段趋势和主要支撑阻力。
- MA144：中长期趋势结构与重要分水岭。
- MA240：年线级牛熊边界和长期安全边际。
- 20日/60日箱体仍用于区间统计；箱体周期不等同于均线周期。

### 1. 个股上涨趋势

特征：

- 股价在 MA25 / MA60 上方，并观察是否位于 MA144 / MA240 上方
- MA25 向上，MA60至少走平；MA144/MA240用于确认中长期结构
- 高点、低点逐步抬高
- 回调缩量，不有效跌破趋势支撑
- 强于所属板块或至少同步板块

操作含义：

- 可作为持有或趋势跟踪对象
- 若日线 J 值回落低位，但趋势未破，可视为趋势内调整
- 若大盘/板块支持，可等待低吸或回踩确认

右侧趋势补充观察：

- 震荡向上结构未破时，重点观察日线 KDJ 的 J 值是否进入大负值或该股历史极低分位。
- 日线 J 大负值说明短期超跌程度加深，属于右侧趋势内的候选观察信号，不单独构成买点。
- 还需等待至少一类修复确认：J 低位拐头、缩量止跌、关键支撑企稳、MACD 绿柱缩短、重新站回短期均线，且板块和大盘不否决。

### 2. 个股下跌趋势

特征：

- 股价在 MA25 / MA60 下方，并观察是否同时跌破 MA144 / MA240
- MA25 向下，MA60走平或向下；MA144/MA240下行代表中长期风险升级
- 高点、低点逐步降低
- 反弹不能站上关键均线
- 弱于所属板块

操作含义：

- 默认不直接做左侧抄底，先判断是否“跌到位”并进入观察区
- 价格从长期高位大幅下跌（例如从 100 元跌到 10 元）只表示跌幅很深，不证明已经见底
- 周线、月线 KDJ 的 J 值跌至负数或历史极低分位，只代表中长期超跌，不代表买点
- 估值必须结合自身历史分位、行业可比、盈利质量和盈利预测可信度判断是否足够低；不能只看静态 PE/PB
- 持仓应进入风控或减仓观察
- 若同时处于 0AMV 活跃实质空头区间，反弹默认是减仓窗口，而不是趋势反转确认
- 需要底背离、放量修复、重新站上均线后才考虑重新评估

左侧趋势的观察升级条件：

1. 相对历史高点或主要波段高点的跌幅已充分释放，并分析下跌原因。
2. 周线或月线 KDJ 的 J 值为负或处于该股历史极低分位。
3. 估值处于自身历史低分位，并排除盈利持续下修造成的“价值陷阱”。
4. 下跌速度放缓，出现底背离、成交量衰竭、关键低点不再创新低等至少一项止跌迹象。

前三项满足只能进入“左侧重点观察”，第 4 项及后续结构修复出现后，才允许生成条件化试探计划。

### 3. 个股横盘震荡

特征：

- MA25 / MA60 走平或缠绕；MA144 / MA240用于识别大级别方向
- 股价在区间内反复波动
- 成交量无持续放大
- 上下沿反复验证

需要识别：

- 箱体上沿
- 箱体下沿
- 箱体中轴
- 当前价格在箱体中的位置

操作含义：

- 接近箱体下沿：观察低吸，但必须结合板块和大盘
- 接近箱体上沿：不追高，观察是否突破
- 放量突破上沿：可能转为上涨趋势
- 跌破下沿：进入风控或止损

## 个股箱体判断

默认周期：

- 短期箱体：20 日
- 中期箱体：60 日

稳健识别：

- 箱体上沿：近 N 日最高价的 80%-90% 分位数
- 箱体下沿：近 N 日最低价的 10%-20% 分位数
- 箱体中轴：上沿与下沿的均值

## 个股 KDJ / MACD 规则

### 日线 KDJ

- J < 12：短期调整到位观察区
- J < 0：短期深度超跌观察；“大负值”优先使用该股自身历史分位判断，不设跨股票统一死阈值
- J 低位拐头：短期修复增强
- J > 90：短期过热
- J 高位钝化：趋势强但不适合盲目追高

### 日线 MACD

- 绿柱缩短：调整动能减弱
- 低位金叉：短期修复
- 底背离：价格新低但 MACD 不创新低
- 顶背离：价格新高但 MACD 不创新高
- 高位死叉：短线风控

### 周线 / 月线

- 周线决定中期趋势是否健康
- 月线决定长期趋势是否仍成立
- 左侧趋势中，周线/月线 J < 0 或处于历史极低分位，可作为中长期跌幅释放的观察证据，但不能单独触发买入
- 周线或月线顶背离，个股即使日线强，也要降低仓位预期

## 重要约束

1. 个股技术调整到位不等于立即买入。
2. 个股买点必须服从板块状态和大盘环境。
3. 在 0AMV 空头区间，J < 12 只能作为观察，不能直接重仓买入。
4. 个股弱于板块时，即使指标低位，也要谨慎。
5. 个股强于板块时，才有资格进入核心观察。
6. 左侧的深跌、周/月线 J 为负和低估值是“跌到位观察证据”，不是自动买点。
7. 右侧日线 J 大负值是“趋势内超跌候选”，必须等待修复信号确认。
8. 0AMV 活跃实质空头区间内，所有反弹优先按减仓机会处理；不得仅凭单日上涨、指标修复或收复一条均线改判为反转。
9. 只有 0AMV 退出实质空头区间，且大盘、板块、个股至少完成相应结构修复确认后，才重新评估持有或买入权限。
10. 长期深跌修复、局部 B1 回调和事件急跌反弹必须分别按“大坑/小坑/小洞”管理，不得混用持有周期与退出规则；完整口径见 `pit_recovery_strategy.md`。
