# 每日持股检查手册

日期：2026-07-14
版本：`holding-check-v1`

## 一、定位

本手册用于每日持仓检查，核心目标是避免把盘中噪声、同步下杀和冲刺阶段的快速洗盘误判为卖出信号。

适用对象：

- 已有持仓，尤其是趋势仍向上、形态仍完整、处于冲刺或加速后的强势股。
- 波段仓的普通持有、减仓、止盈和卖出判断。

不适用豁免：

- 预先定义的硬止损触发。
- 重大风险事件、基本面证伪、流动性恶化或跌停风险。
- 0AMV 活跃实质空头区间内已经制定的反弹减仓规则。
- 单票超 20% 后需要恢复仓位上限的计划性处理。

## 二、盘中噪声过滤

### 1. 忽略普通盘中冲高回落

冲刺阶段个股经常通过盘中快速回落、日内剧烈震荡或一至两日小阴线完成洗盘。只要收盘结构未破，不得仅因盘中冲高回落卖出。

必须先检查：

- 收盘价是否仍维持在关键趋势线或 BBI 线上方。
- 日线趋势是否仍向上。
- 是否出现巨量阴线、放量破位或连续高位放量滞涨。
- J 是否出现有效死叉或趋势动能明显恶化。

### 2. 下杀后看尾盘和次日修复

当天下杀后，必须检查两个修复窗口：

1. 当日尾盘是否快速收回。
2. 次日是否快速收回。

若当日下杀但尾盘快速收回，或次日迅速修复，优先按洗盘或分歧消化处理，不按单日恐慌卖出。

### 3. 不在普通下杀中卖出

没有触发硬止损、重大风险、流动性恶化或预设强制退出条件时，不在快速下杀过程中做情绪化卖出。

普通波段卖出仍以 14:45 后或收盘确认信号为准。

### 4. 大盘突然下杀时保护强势股

若大盘突然下杀，持仓个股同步下跌但此前仍为当日强势或相对强势，不得草率处理。

必须区分：

- 个股自身主动转弱。
- 大盘带动的同步下杀。
- 板块核心股是否同步破位。
- 尾盘或次日是否快速修复。

同步下杀但个股强势结构未破时，优先观察，不直接卖出。

## 三、收盘结构检查

### 1. 缩量小阴线再等一天

若收盘为缩量小阴线，且未跌破 BBI、关键趋势线或结构低点，原则上再等一天确认。

不得把缩量小阴线直接等同于趋势结束。只有次日继续放量下跌、跌破 BBI 或关键结构位，才升级为减仓/止损评估。

### 2. 完美形态不主观卖出

形态完整的持仓，不做主观顶部判断。

若满足：

- 不破 BBI 线。
- 日线趋势仍向上。
- 无巨量阴线或放量破位。
- J 未有效死叉。

则继续持有，不因盘中上蹿下跳或主观担心见顶卖出。

### 3. 多看日线收盘价

每日检查收盘价序列，而不是只看分时波动。

只要日线收盘价仍在抬高，或至少没有破坏上升结构，就不要因盘中波动恐慌处理。

## 四、继续持有的组合条件

以下条件同时成立时，默认结论为“拿住/继续持有”：

```text
收盘价上升或上升结构未破
AND 收盘未跌破 BBI
AND 无巨量阴线或放量破位
AND 日线趋势仍向上
AND J 未有效死叉
```

执行含义：

- 忽略盘中上蹿下跳。
- 不因普通冲高回落卖出。
- 不因缩量小阴线提前卖出。
- 若出现大盘同步下杀，先看尾盘与次日修复。

## 五、升级为卖出/风控评估的条件

出现任一情况时，持有结论失效，进入减仓、止损或清仓评估：

- 收盘有效跌破 BBI，且次日不能快速收回；若连续两日收盘跌破 BBI，按 B1 清仓规则执行。
- 出现巨量阴线、放量破位或高位连续放量滞涨。
- J 有效死叉且价格结构同步转弱。
- 收盘价不再抬高，并跌破上一关键日线低点或 N 型结构低点。
- 板块核心股集体破位，个股也不再强于板块。
- 触发硬止损、重大风险、流动性恶化、跌停风险或 0AMV 空头反弹减仓计划。

## 六、每日输出字段

每日持仓检查必须输出：

| 字段 | 说明 |
|---|---|
| 盘中冲高回落 | 是否发生；是否只是噪声 |
| 当天下杀修复 | 尾盘是否快速收回 |
| 次日修复 | 次日是否快速收回，若尚未发生则列为待验证 |
| 大盘同步下杀 | 是否属于市场带动，个股是否仍相对强势 |
| 收盘K线 | 是否缩量小阴线、巨量阴线或放量破位 |
| BBI | 收盘是否在 BBI 上方 |
| 日线收盘价 | 收盘价是否继续抬高或保持上升结构 |
| 日线趋势 | 是否仍向上 |
| J状态 | 是否有效死叉 |
| 持股结论 | 拿住 / 再等一天 / 减仓评估 / 止损评估 / 清仓评估 |
