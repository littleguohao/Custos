# 大盘回调后的资金切换与标的选择

日期：2026-07-14
版本：`pullback-rotation-v1`

## 一、总逻辑

大盘每一次调整，都可能伴随市场资金在主题、风格、位置和个股之间重新分配。回调后的首要任务不是猜最低点，而是识别资金离开了哪里、保留在哪里、重新流入哪里。

总原则：

```text
弃弱留强
顺势而为
先看主题和风格切换
再按相对强弱选择个股
```

市场回调不是自动买点。只有市场许可、主题结构、板块资金和个股修复共同确认后，才形成新增仓位许可。

## 二、四类标的

### 第一类：逆势继续上涨

特征：

- 大盘调整期间仍继续上涨。
- 收盘价持续抬高，明显强于大盘和所属主题。
- 量价健康，未出现巨量阴线、放量滞涨或高位派发。

结论：相对强度最高，优先保留和进入核心观察池。已有持仓以拿住为主；新增买入仍须等待合适 B1、回踩或其他已定义买点，不追高。

### 第二类：浅跌且守住 BBI

特征：

- 跌幅明显小于大盘或所属主题。
- 收盘保持在 BBI 线上方。
- 日线趋势、关键低点和原持有逻辑未破坏。

结论：趋势未变，已有持仓拿住不动。若收盘为缩量小阴，按每日持股检查手册再等一天，不主观卖出。

### 第三类：同步下跌后缩量快速修复

特征：

- 与大盘同步下跌，但整段跌幅不大。
- 调整过程中成交量迅速收缩，主动抛压减弱。
- 随大盘修复时能够放量快速反弹，并收复 BBI、关键支撑或下杀区间。

结论：趋势尚未破坏，列为等待修复/重点观察。必须看到缩量调整和放量修复的组合证据；只有同步下跌、没有快速修复的，不得提前标记为强势。

### 第四类：深跌、弱反弹或继续创新低

特征：

- 与大盘同步大幅下跌，跌幅显著大于市场或所属主题。
- 大盘反弹时跟随弱、不能快速收复关键位。
- 反弹后继续创新低，或 BBI、关键低点和日线趋势被破坏。

结论：趋势结束或进入高风险状态。已有持仓进入清仓评估；若触发预设硬止损、重大风险或流动性风险，立即执行。普通波段仓在 14:45 后或收盘确认弱反弹、创新低和结构破坏后果断清仓，避免在盘中同步下杀时误卖。

## 三、四个资金切换检查

### 1. 主题是否切换

检查原主题与新主题的相对表现，并在 B1 框架中判断主题走势结构：

- 原主题处于启动、拉升、冲刺、顶部、分歧还是退潮。
- 是否出现顶部巨量阴线、核心股补跌、后排大面积亏损或板块 BBI/关键结构破坏。
- 新主题是否出现成交额、涨停结构、核心股和板块指数共振。

主题结构到顶时，不因个别股票低位 J 值继续沿用原主题买入许可。

### 2. 主题内部是否分化

主题内必须区分龙头、核心、中军、弹性、后排和风险股，并用盘面试盘验证：

- 回调时谁抗跌。
- 修复时谁先放量、先收复关键位。
- 谁在大盘反弹时仍弱或继续创新低。

根据验证结果调整仓位：保留强者，降低后排和弱修复标的，不做平均配置。

### 3. 大小票是否切换

在 B1 候选和主题成分中输出不同市值层的分布比例：

- 大市值/中军候选数量及占比。
- 中小市值/弹性候选数量及占比。
- 两类股票的涨跌幅中位数、上涨家数占比、成交额变化和修复速度。

若资金从小票切向大票，优先中军和容量核心；若从大票切向小票，仍须过滤后排情绪股和流动性风险。

### 4. 高低位是否切换

在 B1 候选和主题成分中输出高位与低位股票的分布比例：

- 高位股、阶段中位股、低位股的候选数量和占比。
- 各位置层的涨停/大涨占比、跌停/大跌占比和修复速度。
- 高位核心是否继续创新高，低位补涨是否形成持续梯队。

高低切换不能只看单只低位涨停；必须看到概念内分布比例和连续性变化。

## 四、决策顺序

```text
确认大盘调整与0AMV状态
-> 检查主题是否切换
-> 检查主题内部是否分化
-> 检查大小票与高低位切换
-> 将个股分入四类强弱梯队
-> 保留第一/第二类
-> 等待第三类确认修复
-> 清理第四类
```

## 五、与每日持股检查的冲突处理

- 盘中同步下杀时，先执行每日持股检查，不在普通下杀中草率卖出。
- 第四类清仓必须以硬止损/重大风险，或 14:45 后、收盘确认的弱反弹、创新低和结构破坏为依据。
- 第一、第二类的“拿住”不豁免 0AMV 空头区间的反弹减仓、单票超 20% 和重大风险规则。
- 第三类只有完成“迅速缩量调整 + 放量快速修复”后才能升级，不能用主观“会修复”代替事实。

## 六、每日输出字段

| 层级 | 必备字段 |
|---|---|
| 大盘 | 调整区间、指数跌幅、0AMV状态、反弹强度 |
| 主题切换 | 原主题阶段、新主题证据、是否到顶/退潮、主题相对强弱 |
| 主题分化 | 龙头/核心/中军/后排、抗跌者、修复领先者、创新低者 |
| 大小切换 | 各市值层数量占比、涨跌幅中位数、上涨家数占比、成交额变化 |
| 高低切换 | 高/中/低位数量占比、涨跌停分布、修复速度、梯队持续性 |
| 个股分层 | 第一/第二/第三/第四类、相对大盘跌幅、BBI、缩量、反弹、是否创新低 |
| 最终动作 | 核心保留 / 拿住 / 等待修复 / 清仓评估 |

在完成跨市场阶段回测前，“跌幅不大”“大幅下跌”“快速修复”必须同时输出实际跌幅、相对大盘差值、观察窗口和量比，不得静默使用未披露阈值。
