# 角色职责调整：产业研究与技术跟踪分工

日期：2026-07-09

## 调整结论

KDJ、MACD、多周期、箱体、上涨/下跌/横盘震荡等技术走势判断，主责应归属于：

- theme_tracker：板块走势与关键节点
- portfolio_review：个股持仓技术状态
- buy_strategy / risk_control：买卖触发和风控执行

industry_research 不应承担技术形态监控主责。

## industry_research 新定位

industry_research 更偏宏观研究、产业研究和长期机会发现。

职责：

1. 宏观政策与产业政策研究
2. 产业趋势判断
3. 技术路线演进
4. 产业链拆解
5. 公司产业地位判断
6. 业绩兑现与景气验证
7. 长期观察池维护

不主责：

- KDJ/MACD 日常监控
- 箱体上下沿计算
- 板块技术突破/破位判断
- 个股买卖点判断

## theme_tracker 新增主责

theme_tracker 除了判断主线题材，还必须负责重点板块技术走势。

包括：

- 板块上涨 / 下跌 / 横盘震荡
- 板块箱体上沿 / 下沿 / 中轴
- 板块日线 KDJ / MACD
- 板块周线 KDJ / MACD
- 板块月线 KDJ / MACD
- 板块关键支撑 / 压力 / 突破 / 破位
- 板块是否调整到位 / 是否见顶 / 是否退潮

## 工作流

industry_research 负责回答：

> 哪些产业值得长期看？为什么？

例如：AI算力、液冷、光模块、半导体、机器人等。

theme_tracker 负责回答：

> 这些产业对应的板块，现在处于什么技术阶段？能不能看？关键节点在哪里？

portfolio_review 负责回答：

> 我持有的个股，结合板块和自身技术状态，今天怎么处理？

## 结论

产业认知决定长期观察方向；
板块技术决定参与窗口；
个股多因子决定具体交易动作。
