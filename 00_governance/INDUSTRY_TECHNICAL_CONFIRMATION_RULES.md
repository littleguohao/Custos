# 产业重点板块技术确认规则

日期：2026-07-09

## 规则定位

产业认知负责判断长期值得跟踪的方向，但是否买入、持有、加仓、减仓，必须结合板块技术状态。

重点产业板块需要每日研判：

- 是否调整到位
- 是否主升延续
- 是否高位钝化
- 是否出现见顶信号
- 是否出现底背离/顶背离
- 日线、周线、月线是否共振

## 适用对象

重点观察板块包括但不限于：

- AI 算力
- AI 服务器
- 液冷散热
- 半导体
- 光模块 / CPO
- PCB / 先进封装
- 存储 / HBM
- 数据中心电源与温控
- 国产算力芯片
- 机器人 / 具身智能
- AI 应用
- 用户持仓所属核心板块

## 技术指标框架

### 1. 日线：短期交易状态

日线用于判断短期是否调整到位、是否过热、是否背离。

重点观察：

- KDJ 的 J 值
- MACD 金叉/死叉
- MACD 红绿柱变化
- MACD 底背离 / 顶背离
- 价格是否跌破/收复关键均线

#### 日线 KDJ 判断

- J < 12：短期进入超跌/调整到位观察区
- J 从低位拐头向上：短期修复信号增强
- J > 90：短期过热，警惕冲高回落
- J 高位钝化：强趋势可继续，但不适合盲目追高

注意：

> J < 12 不是买入信号，只是“调整到位观察信号”，必须结合板块趋势、MACD、成交量和大盘环境。

#### 日线 MACD 判断

- 绿柱缩短：调整动能减弱
- 低位金叉：修复信号
- 价格新低但 MACD 不创新低：底背离观察
- 红柱缩短：上涨动能减弱
- 价格新高但 MACD 不创新高：顶背离风险
- 高位死叉：短线风控信号

### 2. 周线：中期趋势状态

周线用于判断产业板块是否仍处于中期趋势。

重点观察：

- 周线 KDJ 是否在强势区
- 周线 J 是否过热或低位修复
- 周线 MACD 是否在零轴上方
- 周线 MACD 红柱是否扩张或收缩
- 周线是否出现顶背离

判断：

- 周线 KDJ 向上 + MACD 红柱扩张：中期趋势健康
- 周线 KDJ 高位钝化 + MACD 红柱缩短：进入高位分歧
- 周线 MACD 顶背离：中期见顶风险提高
- 周线 KDJ 低位拐头 + MACD 绿柱缩短：中期修复观察

### 3. 月线：长期产业趋势状态

月线用于判断产业方向是否仍处于长期趋势。

重点观察：

- 月线 KDJ 是否上行
- 月线 MACD 是否在零轴上方
- 月线红柱是否持续扩张
- 是否出现长期顶背离

判断：

- 月线 KDJ/MACD 向上：长期趋势仍可跟踪
- 月线高位顶背离：产业趋势可能进入估值消化期
- 月线 MACD 死叉：长期趋势风险显著上升

## 多周期共振规则

### 强势主升

- 日线强
- 周线强
- 月线强

操作含义：

- 可作为核心主线跟踪
- 若 market_timing 支持，可纳入买入/加仓候选

### 短期调整到位，中长期仍强

- 日线 J < 12 或 MACD 绿柱缩短
- 周线/月线趋势未坏

操作含义：

- 进入低吸观察区
- 需要等待日线拐头确认
- 不可在 0AMV 空头区间重仓试错

### 高位见顶风险

- 日线顶背离或死叉
- 周线红柱缩短 / KDJ 高位回落
- 月线高位钝化

操作含义：

- 禁止追高
- 持仓考虑减仓或移动止盈
- 后排股优先退出

### 中长期趋势破坏

- 周线 MACD 死叉
- 月线 MACD 转弱
- 板块跌破关键中长期均线

操作含义：

- 从核心观察池降级
- 持仓进入强风控
- 不再作为中周期主线交易方向

## 每日输出要求

重点产业板块每日必须输出：

| 板块 | 日线KDJ J值 | 日线MACD | 周线KDJ | 周线MACD | 月线KDJ | 月线MACD | 状态 | 操作含义 |
|---|---:|---|---|---|---|---|---|---|

状态可选：

- 主升延续
- 短期调整到位
- 修复观察
- 高位分歧
- 见顶风险
- 中长期转弱

## 重要约束

1. 技术调整到位不等于立即买入。
2. 产业看好不等于无视估值和趋势。
3. 日线信号服务短期交易，周线/月线决定中长期跟踪等级。
4. 当 market_timing 处于防守或 0AMV 空头区间，所有“调整到位”信号只能作为观察，不能直接转为重仓买入。
5. 板块技术状态优先于个股冲动交易。

## 板块日线基本走势判断

在使用 KDJ/MACD 之前，必须先判断板块日线基本走势。

基本走势分三类：

1. 上涨趋势
2. 下跌趋势
3. 横盘震荡

### 1. 上涨趋势

判断条件可包括：

- 收盘价运行在 MA25、MA60 上方，并检查 MA144、MA240 的中长期位置
- MA25 向上，MA60 走平或向上；MA144/MA240不下行时结构更健康
- 高点和低点逐步抬高
- 回调不有效跌破 MA25 或箱体上沿
- 成交量在上涨段放大，回调缩量

操作含义：

- 若 market_timing 支持，可优先关注顺趋势机会
- 若日线 KDJ J 值回落到低位但趋势未破，可视为趋势内调整

### 2. 下跌趋势

判断条件可包括：

- 收盘价运行在 MA25、MA60 下方，并检查是否跌破 MA144、MA240
- MA25 向下，MA60 向下或走平
- 高点和低点逐步降低
- 反弹不能有效站上 MA25
- 下跌放量、反弹缩量

操作含义：

- 原则上不做左侧抄底
- 即使 J < 12，也只代表超跌，不代表趋势反转
- 需要等待趋势修复或底背离确认

### 3. 横盘震荡

判断条件可包括：

- MA25、MA60 走平或缠绕；MA144、MA240用于判断大级别方向
- 价格在一定区间内反复波动
- 上涨和下跌都缺乏持续性
- 高点/低点没有明显单边趋势
- 成交量无持续放大

横盘震荡需要进一步判断箱体：

- 箱体上沿：近 N 日反复受阻的高点区域
- 箱体下沿：近 N 日反复获得支撑的低点区域
- 箱体中轴：上沿和下沿的中间区域

默认 N：20 或 60 个交易日。

### 箱体区间识别方法

#### 简单方法

- 箱体上沿 = 近 N 日高点区域
- 箱体下沿 = 近 N 日低点区域
- 箱体宽度 = 上沿 / 下沿 - 1

若箱体宽度处于合理范围，且价格多次在上下沿之间波动，可视为震荡箱体。

#### 稳健方法

为避免极端影线干扰：

- 箱体上沿可用近 N 日最高价的 80%-90% 分位数
- 箱体下沿可用近 N 日最低价的 10%-20% 分位数

#### 箱体有效性

有效箱体通常满足：

- 至少运行 15-20 个交易日
- 价格至少两次接近上沿或下沿
- MA25/MA60 斜率较小
- 无明显单边突破

### 横盘震荡操作含义

- 接近箱体下沿：观察低吸机会，但必须看大盘和板块情绪
- 接近箱体上沿：不追高，注意减仓或等待突破确认
- 放量突破箱体上沿：可能进入上涨趋势
- 放量跌破箱体下沿：可能进入下跌趋势

## 技术状态判断顺序

板块每日研判顺序必须是：

1. 判断基本走势：上涨 / 下跌 / 横盘震荡
2. 若横盘震荡，识别箱体上沿、下沿、中轴
3. 再看日线 KDJ/MACD
4. 再看周线/月线 KDJ/MACD
5. 最后结合 market_timing 和 0AMV 决定是否可交易
