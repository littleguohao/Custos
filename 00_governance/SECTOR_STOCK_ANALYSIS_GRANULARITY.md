# 板块与个股分析粒度分工

日期：2026-07-09

## 核心原则

板块主要判断：

- 基本走势
- 关键节点
- 趋势阶段
- 是否支持个股交易

个股需要判断：

- 板块归属
- 个股地位
- 相对强弱
- 技术形态
- 量价结构
- 基本面/产业逻辑
- 资金行为
- 风险事件
- 买卖计划

## 板块分析粒度

板块不需要像个股一样拆过多细节，重点回答：

1. 板块处于上涨、下跌，还是横盘震荡？
2. 板块处于启动、主升、分歧、修复，还是退潮？
3. 板块是否强于大盘？
4. 板块关键节点在哪里？
5. 板块是否支持个股买入/持有/加仓？
6. 板块是否发出风险信号？

### 板块关键节点

板块每日重点跟踪：

- 趋势支撑位：MA25、MA60、MA144、MA240、前低、箱体下沿
- 压力位：前高、箱体上沿、跳空缺口、密集成交区
- 突破点：放量突破箱体上沿或前高
- 破位点：跌破 MA25 / MA60 / MA144 / MA240 / 箱体下沿
- 分歧点：放量冲高回落、炸板增加、核心股滞涨
- 修复点：跌破后重新站回关键位，或 KDJ/MACD 修复

### 板块输出应简洁

板块输出不追求复杂评分，重点给结论：

| 板块 | 走势 | 阶段 | 关键支撑 | 关键压力 | 风险信号 | 对个股影响 |
|---|---|---|---|---|---|---|

## 个股分析粒度

个股分析必须更细，因为最终交易发生在个股上。

### 个股多因子框架

| 因子 | 说明 |
|---|---|
| 大盘环境 | market_timing 是否允许交易 |
| 板块状态 | 所属板块是否支持 |
| 个股地位 | 龙头 / 核心 / 中军 / 跟风 / 后排 |
| 相对强弱 | 是否强于板块和大盘 |
| 技术走势 | 上涨 / 下跌 / 横盘震荡 |
| 箱体位置 | 上沿 / 中轴 / 下沿 / 突破 / 跌破 |
| KDJ/MACD | 短中长期技术状态 |
| 量价结构 | 放量突破 / 缩量回调 / 放量滞涨 / 量价背离 |
| 资金行为 | 主力流入流出、龙虎榜、北向/机构持仓等 |
| 基本面/产业 | 是否有真实产业逻辑和业绩验证 |
| 事件催化 | 公告、财报、政策、产业事件 |
| 风险事件 | 减持、解禁、业绩雷、监管、商誉等 |
| 交易计划 | 买入条件、止损、止盈、仓位 |

## 决策差异

### 板块层

板块只决定：

- 能不能看
- 能不能做
- 是顺风还是逆风
- 关键位在哪里

### 个股层

个股决定：

- 买不买
- 买多少
- 哪里买
- 哪里错
- 哪里卖
- 是否进入冷却

## 重要规则

1. 板块支持，不代表所有个股都能买。
2. 板块不支持，个股除非有强独立逻辑，否则不应加仓。
3. 板块只需判断走势和关键节点，个股需要更多细因子。
4. 个股分析必须比板块分析更细。
5. 总控决策中，板块是过滤器，个股是执行对象。
