# market_timing 规则更新记录

日期：2026-07-09

## 本次新增

### 1. 宏观政策环境判断

市场择时必须加入宏观政策背景：

- 货币政策：宽松 / 中性 / 收紧
- 财政政策：积极 / 中性 / 收缩
- 信用环境：扩张 / 稳定 / 收缩
- 监管环境：呵护市场 / 中性 / 压制风险偏好

核心判断：当前市场是否处于“货币宽松 + 财政积极”的政策友好期。

若不是政策友好期，不轻易给出高仓位进攻结论。

### 2. 指南针 0AMV 活跃市值

0AMV 用作 A 股资金风险偏好的关键指标。

规则：

- 0AMV 当日涨幅 > 4%：市场进入做多区间
- 0AMV 当日跌幅 > 2.3%：市场进入空头区间
- 其他区间：中性震荡，需结合指数、情绪、主线判断

动作：

- 做多区间：可上调市场状态一级，但仍需确认主线和情绪不背离
- 空头区间：必须提高风控等级，原则上禁止追涨和扩大短线仓位
- 与指数背离时，优先把 0AMV 作为资金风险偏好信号

## 权重调整

旧框架：指数趋势、市场宽度、情绪强度、成交量能、主线清晰度。

新框架：

| 模块 | 权重 |
|---|---:|
| 宏观政策环境 | 15 |
| 0AMV 活跃市值 | 15 |
| 指数趋势 | 20 |
| 市场宽度 | 15 |
| 情绪强度 | 15 |
| 成交量能 | 10 |
| 主线清晰度 | 10 |

## 影响

market_timing 不再只是技术面/情绪面判断，而是升级为：

宏观流动性 + 资金风险偏好 + 指数趋势 + 市场情绪 + 主线结构 的综合择时系统。

## 2026-07-09 补充：大盘定义

用户确认：market_timing 中的“大盘”必须同时包括：

- 上证指数
- 创业板指数
- 科创板指数/科创50
- 北证50

不能只用上证指数代表大盘。

若四类指数明显分化，market_timing 必须输出“结构性行情”判断，并说明哪类资产强、哪类资产弱。

## 2026-07-09 补充：外围市场影响

用户补充：美股、日股、韩股波动对中国股市的影响也需要纳入 market_timing。

已新增“外围市场影响”模块，权重 10%。

重点观察：

- 美股：纳斯达克、标普500、费城半导体指数、AI/科技龙头
- 日股：日经225、半导体设备、电子材料、机器人方向
- 韩股：KOSPI、三星电子、SK海力士、存储/HBM/半导体链
- 港股：恒生指数、恒生科技指数、中概科技股

规则：

- 外围科技强，有利于 A 股 AI/半导体/光模块/PCB/消费电子等风险偏好
- 外围科技弱，A 股科技成长开盘承压，降低追高权限
- 外围强但 A 股不跟，说明内资不认可，不能单独作为加仓依据
- 外围弱但 A 股抗跌，说明内资强势，可提高相关板块相对强度评价

注意：外围市场只能作为风险偏好和方向映射信号，不能替代 A 股自身 0AMV、宽度、情绪和主线确认。

## 2026-07-09 补充：外围市场数据来源

用户询问外围市场是否可通过华尔街日报等财经媒体获得。

结论：可以，但财经媒体更适合作为解释来源，结构化行情数据更适合自动评分。

原则：

- 行情涨跌幅优先使用结构化来源，如 Yahoo Finance、MarketWatch、Investing、Nasdaq、TradingView 等。
- WSJ、Reuters、Bloomberg、CNBC 等用于解释市场波动原因和风险事件。
- WSJ 可能存在付费墙，不作为唯一自动化数据源。
- 不复制长篇付费内容，只提炼事实和市场影响。
